Dokumentasi API v1

Integrasikan pengunduh TikTok tanpa watermark ke dalam aplikasi Anda dengan mudah menggunakan Proxy API kami.

Main Endpoint

Metode: GET

GET https://tikdown.ikyzxz.my.id/api/v1?url={TIKTOK_URL}
✓ Link berhasil disalin!

📝 Query Parameters

  • url Required (Link TikTok)

🛡️ Keamanan & Limit

Batas aman: 30 request / 10 detik. Tidak perlu mengirimkan parameter Fingerprint, server kami akan menanganinya secara otomatis.
